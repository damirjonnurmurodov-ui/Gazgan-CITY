# Gazgan City Flutter UI Skill

## Purpose

Build and review the Gazgan City Flutter UI prototype using only the approved project instructions and UI PNG prototypes.

## Always read

- `AGENTS.md`
- `RESTRICTIONS.md`
- relevant `screens/*.md`
- `UI_REFERENCE_GUIDE.md`

## Never add

- Gazgan Invest
- marmar products
- investment features
- ROI calculator
- backend
- Supabase
- Firebase
- real Google Maps
- auth
- payment

## Workflow

1. Inspect current files.
2. Make a concise plan.
3. Edit only relevant files.
4. Run `flutter analyze`.
5. Report changed files and verification.

## UI rules

Follow `ui_images/` prototypes exactly in style, spacing, color, and mobile layout.
