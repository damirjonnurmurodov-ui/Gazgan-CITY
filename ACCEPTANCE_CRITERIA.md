# Acceptance Criteria — Gazgan City UI Prototype

## Overall

Prototype qabul qilinadi, agar:

- 5 asosiy bo‘lim ishlasa;
- UI PNG prototiplarga yaqin bo‘lsa;
- no overflow;
- no clipped text;
- SafeArea ishlatilgan bo‘lsa;
- bottom nav kontentni yopmasa;
- Gazgan Invest yoki marmar/invest bo‘limlari bo‘lmasa;
- `flutter analyze` jiddiy xatosiz o‘tsa.

## Screens

### Home

- Header, badge, weather, official banner, quick actions, receptions, news, useful sections, local services mavjud.

### Map

- Search, chips, mock map, markers, selected place card, quick places mavjud.

### Listings

- Search, filter, category menu, safety banner, listing cards mavjud.

### News

- Category chips, featured news, news list, official source banner mavjud.

### Profile

- Profile header, avatar, menu items, official support card mavjud.
