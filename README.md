# Gazgan City

**Gazgan City** — G‘ozg‘on shahri aholisi uchun rasmiy shahar xizmatlari mobil ilovasi.

## Shior

**G‘ozg‘on hayoti bitta ilovada.**

## Maqsad

Ilova shahar yangiliklari, rasmiy e’lonlar, sayyor qabullar, mahalliy e’lonlar, xarita, taxi, ustalar, korxonalar, ish o‘rinlari va foydali kontaktlarni bitta qulay platformaga jamlaydi.

## Asosiy bo‘limlar

1. Bosh sahifa
2. Xarita
3. E’lonlar
4. Yangiliklar
5. Profil

## Joriy bosqich

Hozirgi bosqich: **Flutter UI prototip**.

Hozircha qo‘shilmaydi:

- backend;
- Supabase;
- Firebase;
- login/auth;
- real Google Maps;
- to‘lovlar;
- push notification.

Mock/static data ishlatiladi.

## Muhim cheklov

Gazgan City loyihasiga **Gazgan Invest qo‘shilmaydi**.
Marmar mahsulotlari, investitsiya loyihalari, ROI kalkulyator, marmar savdo, marmar eksport, marmar katalog kabi bo‘limlar qo‘shilmaydi.

## Dizayn manbasi

Dizayn uchun yagona vizual manba — `ui_images/` papkasidagi tasdiqlangan Gazgan City PNG prototiplar.
Tashqi ilovalar yoki boshqa loyihalar namuna sifatida ishlatilmasin.
