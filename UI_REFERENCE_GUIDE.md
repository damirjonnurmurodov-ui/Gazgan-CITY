# UI Reference Guide

## Yagona dizayn manbasi

Flutter UI faqat `ui_images/` papkasidagi tasdiqlangan PNG prototiplarga asoslanadi.

## PNG fayllar

- `00_full_ui_prototype_preview.png` — barcha ekranlar umumiy ko‘rinishi
- `01_home_screen.png` — bosh sahifa
- `02_map_screen.png` — xarita
- `03_listings_screen.png` — e’lonlar
- `04_news_screen.png` — yangiliklar
- `05_profile_screen.png` — profil

## Agent uchun qoida

- PNGdagi umumiy spacing va layout saqlansin.
- Har bir ekran 9:16 mobil formatga mos bo‘lsin.
- Matnlar xira yoki mayda bo‘lmasin.
- UI elementlar o‘zaro ustma-ust chiqmasin.
- Bottom navigation kontentni yopmasin.
- Tashqi appga taqqoslama berilmasin.
